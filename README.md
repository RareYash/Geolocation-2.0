**Step 1 — Clone the repo**
*(Nobody:  ...   Absolutely nobody:  ...   You:)*
```bash
git clone https://github.com/your-username/city-locator.git
cd city-locator
```

**Step 2 — Virtual Environment**
*"You either die using system Python, or live long enough to use a venv."*
```bash
python -m venv venv

# Windows warriors 🪟
.\venv\Scripts\activate

# Linux/macOS chads 🐧
source venv/bin/activate
```

**Step 3 — Install Dependencies**
*pip install, they said. It'll be fun, they said.*
```bash

```

**Step 4 — Run It**
